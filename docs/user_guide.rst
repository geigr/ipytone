
User Guide
==========

This user guide contains a few tutorials covering the main features of ipytone.
Do not hesitate to also check the Tone.js `Documentation`_ and `Wiki`_, which
both contain useful information. Ipytone works very much like Tone.js and copies
its `API`_ in Python.
